# Sierpinski-Triangle

![Sierpinski-Triangle image](https://github.com/NisalaTharakaPerera/Sierpinski-Triangle/blob/main/SierpinskiTriangle.png)

#### First Transformation
$$x_{n+1}=0.5x_n$$  
$$y_{n+1} = 0.5y_n$$ 

#### Second Transformation
$$x_{n+1} = 0.5x_n+ 0.5$$  
$$y_{n+1} = 0.5y_n+ 0.5$$  

#### Third Transformation
$$x_{n+1} = 0.5x_{n+1}$$  
$$y_{n+1} = 0.5y_n$$
